ParentFont Property

Applies to

THtmlViewer component

Declaration

property ParentFont: Boolean;

Description

The ParentFont property decides which font to use if no font information is in the document.

If ParentFont is true then the Font property of the parent component is used else the font is defined by properties DefFontColor, DefFontName and DefFontSize.
