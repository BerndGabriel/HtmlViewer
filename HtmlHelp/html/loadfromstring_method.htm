LoadFromString Method

Applies to

ThtmlViewer component

Declaration

procedure LoadFromString(const S: string; const Reference: string = ''); overload;

procedure LoadFromString(const WS: WideString; const Reference: string = ''); overload; {Restrictions}

Description

The LoadFromString method loads an HTML document from a string.

Reference is an optional reference parameter.

Restrictions for Overloaded WideString Version

Available only with Delphi 6 and later.

This version of LoadFromString is for Unicode (UTF-16) input only.

The FindDisplayPos and FindSourcePos methods do not work with Unicode input.
