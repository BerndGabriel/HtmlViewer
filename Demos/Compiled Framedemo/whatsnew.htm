Version 11.3

Added
  • Issue 127: Embedded Images (image URL starts with 'data:image/', continues with ';base64,' and a lot of gibberish).
  • HTML5 Tags <header>, <section>, <nav>, <article>, <aside>, <footer>, <hgroup>, and <mark>.
  • Property TViewerBase.QuirksMode, which can be one of qmDetect, qmStandards, and qmQuirks.

    Implemented the following in Quirks mode:

    • List Item Markers
      The default value of list-style-position is "inside" if the list-item is outside a <UL> or <OL> element, and "outside" otherwise.
      Documented at http://msdn.microsoft.com/en-us/library/gg558047%28v=vs.85%29.aspx
    • Table Font Inheritance
      With the exception of font-family, by default table elements do not inherit font properties.
      Documented at http://msdn.microsoft.com/en-us/library/gg558047%28v=vs.85%29.aspx and verified by J. Peter Mugaas.
    • Opera Quirk
      Font sizes are increased by one and the default font size is small.
      Documented at http://www.opera.com/docs/specs/doctype/
    • Box-Sizing Model
      Is content-box instead of border-box.
      Described by http://www.quirksmode.org/css/box.html and by Microsoft at http://msdn.microsoft.com/en-us/library/gg558047%28v=vs.85%29.aspx
    • Aligned Images
      Right-aligned images have a default left margin of 3 pixels. Left-aligned images have a default right margin of 3 pixels.
      Described at http://www.cs.tut.fi/~jkorpela/quirks-mode.html and in Mozilla's quirks.css file.

  • New Demo FrameBrowserUsingICSv7: Same as FrameBrowserUsingIndy but using Overbyte's ICSv7 instead of Indy (Angus Robertson)
  • Some support for CSS3 colors RGBA (Red, Green, Blue, Alpha), HSL (Hue, Saturation, Lightness), and HSLA (Hue, Saturation, Lightness, Alpha). (J. Peter Mugaas)
  • CSS properties in standards (non quirks) mode: CSS3 box-sizing, CSS2 max-height, CSS2 min-height, CSS2 max-width, CSS2 min-width (J. Peter Mugaas)
  • Delphi XE2+ only: support for Vcl.Themes StyleService colors using the StyleServices.GetSystemColor call. (J. Peter Mugaas)

  • HandleMeta and InitLoad promoted to virtual, descendents can now do something. For example Sérgio Alexandre uses meta tags to define base64 encoded images: <meta name="cid" HttpEq="image/*" content="base64">, so any <img src="cid"> will display that image. (Sérgio Alexandre)

  • BeginProcessing, EndProcessing promoted to virtual, same purpose like InitLoad. (Sérgio Alexandre)
  • Since "GetViewerClass" is virtual, "CreateViewer" is now virtual. E.g. Sérgio Alexandre uses it to remove borders. (Sérgio Alexandre)
  • GDIPL2A.pas - Loading png-files no longer produces temporary files. Using IStreams instead.
  • Property TViewerBase.UseQuirksMode replaces the definable symbol "Quirk". THtmlViewer now can decide to use the quirks mode depending on document. (J. Peter Mugaas)
